# Sales Data Analysis using Microsoft SQL Server


## Project Description
This project focuses on analyzing sales data using Microsoft SQL Server.  
The dataset consists of customer, product, and sales tables stored in CSV format and imported into SQL Server for analysis.


## Dataset
The project uses three tables:
- dim_customers – contains customer information
- dim_products – contains product details
- fact_sales – contains sales transactions
